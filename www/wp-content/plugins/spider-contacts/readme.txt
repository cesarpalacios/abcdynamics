== Installation ==

Perform a new installation for the Spider Contacts.

1. Log in to the administrator panel for installing the Spider Contacts
2. Go to Plugins Add > New > Upload.
3. Click "Choose file" ("Browse") and select the Spider Contacts zip file.
4. Click "Install Now" button.
5. If the installation of the contacts succeeded you will see the message in the picture. If any problem occurs during the installation of the contacts, please contact us info@web-dorado.com.
6. Click "Activate Plugin" button for activating Spider Contacts plugin.


== Frequently Asked Questions ==


= How can I create categories for WordPress Contacts? =

On the left menu select Spider Contacts > Categories.

From Spider Contacts Manager choose "Categories". In the upper right corner click on the "Add a category" button.

  * Name. Add a name for the category for the Spider Contacts.
  * Description. Add a description for the category for the Spider Contacts.
  * Parameters. Here you can define different parameters for the category. You can add as many parameters as you wish. After filling in the current parameter title, press "Enter" to proceed to the next parameter.
  * Order. Choose the order of the category for the Spider Contacts.
  * Published. Choose whether to publish the category or not.
    
= How can I create or edit contacts? =
    
On the left menu select  Spider Contacts > Contacts. In the upper right corner click on the "Add a Contact" button.

  * First Name. Add the first name of the contact.
  * Last Name. Add the last name of the contact.
  * Email. Add the Email.
  * Send Email when Message Sent. Choose whether the submitted messages are redirected to the contacts e-mail or not.
  * Category. Choose a category from the existing list of categories.
  * Images. You can add an unlimited number of images to the contact.
  * Description. Add a description for the contact.
  * Order. Choose the order of the contact.
  * Published. Choose whether to publish the contact or not.
    
= How can I view or edit messages submitted to contacts? =

On the left menu select Spider Contacts > Message. In this section you can view all the submitted messages of the Spider Contacts.
	
Select a category for the message of the Spider Contacts. Select a contact for the message of the Spider Contacts.

Each message of the Spider contacts in the list has the following parameters:
  * Message title,
  * Sender name,
  * Phone,
  * Sender,
  * Contact Preferences with Sender,
  * Category,
  * Date.

It is possible to filter the messages of the contacts by the date and name parameters.
    
= How can I edit Global Options of the Spider Contacts? =

Enable/Disable Search Contact on Front-end by Category. Here you can enable or disable searching contacts by category on front-end.

Enable/Disable Search Contact on Front-end by Name. Here you can enable or disable searching contacts by category on front-end.

= How can I publish the created WordPress Contacts ? =        
    
To insert a contact into a Page or a Post:

Press the button named Insert Spider Contacts in a post or a page you want to show the contact.		

**Single Contact**	
Select Contact. Select a contact from the given list.			

**Contacts List Category**
Select Category. Select a category from the given list.	
Contacts display type. Choose whether to display the contacts in a short view, table view or Full view 

If there is any problem with editor, you can add the short code manually to the pages or posts.			

**Single Contact view**

Add the short code [Spider_contact_single id="my id"] to the pages or posts and change the following point.
  * ID. Add the ID (Spider Contacts>Contact >ID) of the single contact to the appropriate place("my id") in the shortcode.	

**Contacts List Category**

Add  the short code  [Spider_contact_categories id="Contact ID" type=""] to the pages or posts and change the following points.	
  * ID. Add the ID (Spider Contacts>Contacts >ID) of the category which you want to show to the appropriate place("my id") in the shortcode. If the value of the ID is "All_CAT", the all categories will be shown.	
  * Type. If the value of the type is 0, the contacts list category will be shown in short view. If the value of the type is 1, the contacts list  category  will be shown in a table view. If the value of the type is 2, the contacts list category will be shown in a Full view.
